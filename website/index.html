poke-dl

GitHub

Download Pokémon Episodes Instantly

A free, open-source tool to download full Pokémon episodes from Pokéflix m3u8 links in my api using FFmpeg and yt-dlp.

Download View README
Made with Python

Why I made This

  1. i read that pokemon tv was shutting down because of cost issues, i saw that their wasn't a free and ad-free way to watch pokemon content anymore.
  2. i was mad, so i took matter to my own hands, and made poke-dl, it uses an api i made, so it will never stop working !!Make sure to give credit to the people who made ffmpeg
  3. poke dl is free, never pay money for it, its also open src

Features: Supports all seasons, episodes, and series. JSON-based episode list is auto-updated, from my repo, no need to update every 2 minutes.

Enjoy poke-dl | Not affiliated with Nintendo or Pokémon Company.
